A modern, responsive **Landing Page** built with **HTML**, **CSS**, and **Bootstrap**.  
Designed to be clean, fast, and easily customizable for any business, startup, or personal brand.

---

## 🧩 Features
- Fully **responsive** layout using Bootstrap grid system  
- **Clean and modern** design  
- **Reusable components** (Navbar, Hero section, Footer, etc.)  
- **Optimized** for mobile and desktop devices  
- Easy to customize colors, text, and images

---

## 🛠️ Technologies Used
- **HTML5**
- **CSS3**
- **Bootstrap 5**
